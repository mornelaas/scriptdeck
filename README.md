# ScriptDeck

It's a content creator tool to help managing scripts as they move in their cycle.

## Tech Stack

- Python 3.14+
- SQLite

## The script lifecycle

A script moves through these states:
1. idea
2. borrador
3. listo
4. grabado
5. publicado

## Project status

Week 1 of the roadmap, bulding the SQLite layer, early development.
